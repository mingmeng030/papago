## 👩🏻‍💻컨벤션 (Convention)

### Commit 전략 

|  **항목**  |             **설명**              |
| ---------- | ---------------------------------- |
| ADD        | (새로운) 기능 추가                 |
| UPDATE     | 기능 수정 or  코드 리팩토링        |
| BUGFIX     | 버그 or 이슈 수정                  |

- 예시 : `git commit -m "ADD : header and footer ui, webpack file-loader"`

## Frontend

### 🔥 기술 스택

|  **항목**  |  **기술 스택**                   |
